This is a stand-in home page for a real templated FastAPI app. The Bug-Fab floating action button (the small bug icon in the bottom-right corner) is rendered by a single <script> tag in base.html and shows up on every page that extends the base template.

To file a report:

  1. Click the bug icon.
  2. Optionally draw on the captured screenshot.
  3. Type a title (and an optional description / severity).
  4. Submit.

The report is stored in SQLite at ./data/bug-fab.db with the screenshot PNG written to ./data/screenshots/. Browse submitted reports at /admin/bug-reports.

This example is deliberately minimal — for production, see DEPLOYMENT_OPTIONS.md in the Bug-Fab repo for the auth, rate-limit, and CSP checklists.
